Land Cover Modeling

Land cover modeling is conducted using up to three stages.

  1. pixel-level unsupervised classification
  2. segment level classification and regression tree
  3. rule-based class adjustments

For the 2006-2009 and 2009-2011 time intervals, only the first stage (the pixel-level classification) was employed and only for the 2009 and 2011 image dates. The 2006 data lacked a near infra-red image band, which is important for land cover modeling.
